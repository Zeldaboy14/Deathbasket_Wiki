Ocarina of Time instrument set editor (OoT InstEd)

Started as a way of helping me create custom instrument sets, this has evolved in to a mostly complete tool for hacking the music in Ocarina of Time.

Progress:
There is not really anything more planned for this tool, but I will be trying to identify and fix any bugs when I can be bothered.

Current Features
  • Save/load instrument set presets
  • 57 instrument presets (most instruments from the game), user can also define their own instruments
  • Import instrument sets into a debug ROM
  • Instrument set to sequence mapping editor
  • Sequence converter
  • Sequence inserter
  • Sample inserter
  • Instrument ripper (OoT debug, MM (U) decompressed, SM64 (U))
Planned Features
  • Zelda sequence to MIDI conversion, though I may make a separate tool for that.
There are a few other things I want to do with this too, if I ever get around to it.
