Functions found manually

Copy from ROM

0x80025C50(u32 rom_addr, u32 ram_addr, u32 size)
Copies size bytes of data from rom_addr to ram_addr

Alternatively,
0x80068A50(u32 rom_addr, u32 ram_addr, u32 size)
copies and decompresses data, used for ex. ground maps

Get switches/variables

0x800299F4(u32 n)
Returns the state of switch number n as used by the SRAM editor debug menu

Audio functions

0x80050BF0(u32 id, u32 ?, u32 ?)
Plays sound given by id

0x80051C08(u32 n, u32 ?)
Plays music sequence n


Text printing functions

0x80074CA8 set_colour(u32 addr2, u32 r, u32 g, u32 b)
Sets text colour

0x80074D0C(u32 addr2, u32 x, u32 y, u32 addr)
0x80088A90(u32 addr2, u32 x, u32 y, u32 addr)
Draws a string at position (x,y) on the screen

0x800AADA0(u32 str_addr)
Returns the length of string at str_addr

0x800AADF0(u32 addr, u32 str_addr, u32 values...)
Sets the text string to be drawn

0x802504E0()
Used for printing debug messages (not functional)

Maths functions

0x800A7B20 cos(f32 x)
0x800AA730 sin(f32 x)
Compute the cosine/sine of angle x, which is given in radians

0x800B3220 random()
Returns a random 16-bit integer

Gameplay functions

0x80032E60(?)
Returns a pointer to ground type data (TODO: write up) for current lie

0x80043A04(u32 x, u32 z)
Returns the y co-ordinate (height) of the ground at position (x,z)

0x800448E0(u32 n)
Returns a pointer to the stats of club n (TODO: write those up)

Menu functions

0x801F5C24(u32 x)
Shows unlock animation for x, where x is:
0     dk
1     maple
2     metal mario
3     koopa park
4     shy guy desert
5     yoshi's island
6     boo valley
7     mario's star

Hole loading functions

0x80068F4C()
"Randomises" the pin location

0x802504E8(u32 course, u32 hole)
Loads hole from course

0x80251DFC(u32 course, u32 hole, u32 component)
Loads component of hole from course. Values of component are as follows (file extensions found in debug messages):
1     geometry 1     (.vtx)
2     geometry 2     (.dtx)
3     trees          (.tre)
4     water          (.tri)
5     ground map     (.att)
Info on these file formats: soon on a new page
